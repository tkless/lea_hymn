Head Mounted Displays

Ein Head Mounted Display besteht aus zwei Monitoren die vor den Augen des Benutzers installiert werden. Spezielle Optiken erlauben dem Benutzer, auf den Miniaturbildschirm zu schauen.Üblicherweise enthalten die HMDs Kopfhörer, so dass der Benutzer in der virtuellen Welt auch hören kann. Außerdem besitzen sie meist einen Head Tracker. Dieser Bestimmt die Position und die Orientierung im Raum. Wenn der Benutzer sich umschaut wird diese Information ständig zum verarbeitenden System übertragen. Dieses berechnet dann die passende Ansicht, die der Benutzer in der Virtuellen Umgebung sehen sollte. Diese wird dann auf den Miniatur-Displays dargestellt.
