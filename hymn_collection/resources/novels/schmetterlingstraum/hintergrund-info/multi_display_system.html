Multi Display System

From IThinkHere

Ein Multi-Display-System dient zur Erweiterung des dem Benutzer zur Verfügung stehenden Sichtbereichs. Bei Verwendung eines Projektors ist zwischen Front- und Rückprojektion zu unterscheiden. Viele Simulatoren nutzen das Vorhandensein mehrerer Displays zur Darstellung unterschiedlicher Blickrichtungen (vorne, hinten, links, rechts). Ist der Benutzer völlig von den darstellenden Geräten (Leinwände) umgeben, spricht man von einer CAVE (CAVE Automatic Virtual Environment). Eine weitere Möglichkeit ist die Projektion in eine Kuppel (könnte man allerdings als grenzwertig betrachten...ist ja nicht wirklich Multi-Display, oder?). Ein großer Vorteil von Multi-Display-Systemen (besonders im größeren Sinne, also wie bei einer CAVE) ist die Tatsache, dass der Benutzer die Umgebung wie in der realen Welt betrachten kann, indem er sich einfach ganz normal umsieht.
