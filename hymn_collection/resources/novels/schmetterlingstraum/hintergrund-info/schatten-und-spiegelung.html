Schatten/Spiegelung eines realen Objekts auf einem Virtuellen Objekt

Wenn ein reales Objekt einen Schatten/eine Spiegelung auf ein Virtuelles Objekt wirft, wird der Schatten/die Spiegelung des realen Objekts am Computer nachmodelliert.

Schatten/Spiegelung eines virtuellen Objekts auf einem realen Objekt

Das reale Objekt wird am Computer nachmodelliert. Das nachmodelierte Objekt wird unsichtbar und dessen Schatten/Spiegelung wird mittels Bildsynthese (z.B.: Ray Tracing) auf dem virtuellen Objekt erstellt.

Schatten/Spiegelung eines virtuellen Objekts auf einem virtuellen Objekt

Der Schatten/Die Spiegelung des virtuellen Objekts wird mittels einer Bildsynthese (z.B.: Ray Tracing) modelliert.
