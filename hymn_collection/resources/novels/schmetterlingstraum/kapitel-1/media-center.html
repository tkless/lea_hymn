Media-Center der Zukunft


Features:

Unter dem Media-Center ist, wie der Name schon verrät, ein Gerät oder eine Einheit zu verstehen welches die verschiedenen Medien vereinen soll. Diese Medien sind:


  • PC / Internet

  • TV / Video

  • Radio/Musik

  • Telefon / E-Mail


Diese Medien sollen wie bereits erwähnt in dem Media-Center zusammen gefasst werden, welcher sich wiederum durch besonders einfache Bedienung auszeichnet.

Die Ausgabe soll bei den visuellen Medien wie heutzutage auf einem in der Zukunft gewöhnlichen Bildschirm, also einem TFT-Display erfolgen. Das bedeutet, dass der Fernsehempfang und das Internet auf ein und dem demselben Gerät ausgegeben werden.

Die Ausgabe der akustischen Medien wird wie beim Musik hören zusätzlich zum Display über ein ganz normales Lautsprechersystem oder beim Telefonieren optional über dieses Lautsprechersystem im Sinne einer Freisprecheinrichtung oder, wie üblich, über den Lautsprecher des Telefons ausgegeben.


Bedienung:

Die Bedienung des Media-Centers erfolgt über ein Hauptgerät, das im Normalfall das Handy des Benutzers sein wird, vorausgesetzt es verfügt werksseitig über diese Option. Notwendig wird dazu eine Software sein, die ganz aufwandslos auf dem Mobiltelefon installiert werden muss und im Lieferumfang des Mediacenters enthalten ist. Eine andere Möglichkeit der Bedienung stellt eine optional erwerbbare Fernbedienung oder eine ebenfalls aufpreispflichtige Sprachsteuerung, welche die Bedienung noch zusätzlich vereinfacht dar.

Jedes Bedienelements, also Mobiltelefon oder Fernbedienung muss bevor es seine Funktionen erfüllen kann vom Administrator des Media-Centers registriert und authorisiert werden.

Jeder Benutzer benötigt eine eigenes Profil über das ihm seine Rechte vergeben werden.

Es sind dann nur noch wenige Tastendrücke / Berührungen auf dem Touchscreen des Bedienelements oder vorgegebene Sprachbefehle zu tätigen, um die jeweils gewünschte Funktion des Media-Centers auszuführen.


  • PC / Internet:

    Der Media-Center ist verfügt über die selben Funktionen wie ein handelsüblicher PC und ist demzufolge genauso zu bedienen.

Die Handhabung erfolgt wie gewohnt über eine Art Mauszeiger, die ähnlich des

Wii-Prinzips über das Handy oder eine Fernbedienung funktioniert. Erforderlich dafür ist eine Sensorleiste, die mittig unter oder über dem Display angebracht werden muss. Texte oder beispielsweise URLs zur Benutzung des Internets können über eine anschliessbare Funktastatur, das Bedienelements oder ganz bequem über Spracheingabe eingegeben werden.

Für PC-Spiele wird es auch möglich sein handelsübliche Gamecontroller zu benutzen, deren Anschluss über USB an der Frontseite des Media-Centers erfolgt.


  • TV / Video:

    Der Media-Center stellt gleichzeitig einen TV-Receiver dar und bietet dem Benutzer

    die üblichen Fernsehmöglichkeiten. Der TV-Empfang erfolgt über das Internet und erlaubt es dem Benutzer ebenfalls Sendungen aufzunehmen, die dann auf dem lokalen Datenträger des Media-Centers gespeichert werden. Andere Video-Dateien können auch bequem von einem anderen Datenträger, sofern dieser per USB mit dem Media-Center verbunden ist, gesehen werden. Die Senderwahl erfolgt über On-Screen-Display und wird entweder durch das Bedienelements oder per Sprachbefehl bestätigt. Ebenfalls ermöglicht der Media-Center eine Alterskontrolle über das Profil des jeweiligen Benutzers und kann so altersgerechte Unterhaltung gewährleisten.

  • Radio / Musik:

    Die Wiedergabe von Musik oder anderen akustischen Medien erfolgt entweder über das Internet oder durch andere Audio-Dateien, die entweder auf dem lokalen Datenträger des Media-Centers oder auf einem anderen mit dem Media-Center verbundenen Datenträger liegen. Radiosender können per Internetstream empfangen werden und selbstverständlich auch gespeichert werden um das Radiohören auch ohne Inbetriebnahme des Displays zu ermöglichen. Für die Wiedergabe von lokalen oder externen Medien erfolgt über OSD. Die Ausgabe findet auf dem angeschlossenen Lautsprechersystem statt.

  • Telefon / E-Mail:

    Der Media-Center unterstützt ebenfalls Telefonie sowie das Versenden und Empfangen von E-Mails. Das Telefonieren wird durch die VoIP-Technologie möglich. Innerhalb der Reichweite des Media-Centers, auf dem das Mobiltelefon registriert ist, wird dann VoIP genutzt. Ausserhalb der Reichweite wird der übliche Netzanbieter genutzt. Bei installierter Sprachsteuerung wird das Handy nicht mehr zum Telefonieren benötigt. Das Telefonieren erfolgt dann über eine Freisprechanlage. Das Verfassen und Versenden von E-Mails geht entweder über die Tastatur des Bedienelements oder über Spracherkennung vonstatten. Empfangene E-Mails werden auf dem Handy oder dem Display des Media-Centers angezeigt oder bei einer installierten Sprachsteuerung optional vorgelesen und über ein angeschlossenes Lautsprechersystem ausgegeben.



Hauptgerät:

Der Media-Center besteht aus einem kompakten Gehäuse, welches in der Regel handelsübliche PC-Hardware-Technologie beinhaltet. Die verwendete Hardware ist herstellerabhängig. Aufrüstmöglichkeiten stellen somit kein Problem dar. Die verwendete Software ist ebenfalls herstellerabhängig, lediglich die Wahl der Software im PC-Betrieb steht dem Benutzer frei. Der Media-Center muss, um dem Benutzer sein volles Funktionsspektrum bieten zu können, mit dem Internet verbunden werden.


Systemvorrausetzungen:

  • funktionsfähiges Display

  • funktionsfähiges Lautsprechersystem

  • vorhandener Internetanschluss

Im Lieferumfang ist enthalten:

  • alle zum Anschluss erforderlichen Verbindungskabel

  • die Software zur Verwendung des Mobiltelefons als Bedienelement, welche auf allen fähigen Mobiltelefonen markenunabhängig funktioniert

  • eine Sensorleiste, die mittig unter oder über dem Display zur Verwendung des Bedienelements angebracht werden muss


Zubehör:

Optional erwerbbar ist eine Sprachsteuerungs-Paket bestehend aus:

  • der Software (zur Installation auf dem Media-Center und dem Mobiltelefon)

  • ein Mikrofon, das frei im Raum angebracht werden kann

  • Voraussetzung: funktionsfähiges Mobiltelefon als Bedienelement


Desweiteren sind alle üblichen Geräte, die mit einem normalen PC kompatibel sind, über die USB-Frontanschlüsse in das System integrierbar.
